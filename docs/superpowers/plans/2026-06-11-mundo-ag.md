# Mundo AG Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transformar o fork do folio-2025 (MIT) na experiência 3D "Mundo AG" da Auto Genius, publicada em `useautogenius.com/mundo/`.

**Architecture:** O jogo vive em `mundo-src/` (projeto Vite independente); o build estático é commitado em `mundo/`. O site principal (`index.html`) só ganha um link de menu. Conteúdo do jogo é data-driven (`sources/data/*.js`), áreas são modulares (`sources/Game/World/Areas/`), cores são parametrizadas.

**Tech Stack:** Three.js (WebGPU/WebGL2 + TSL), Rapier (WASM), GSAP, Howler, Vite. Sem framework de testes — verificação é por build + inspeção visual no dev server + auditorias via grep (comandos exatos em cada task).

**Spec:** `docs/superpowers/specs/2026-06-10-mundo-ag-design.md`

**Convenções deste plano:**
- Diretório de trabalho: `/Users/yurimatos/auto_genius` salvo indicação
- "Verificar no dev server" = `cd mundo-src && npm run dev` + abrir no browser (ou preview tools) e checar console sem erros
- Commits frequentes, mensagens em inglês seguindo o padrão do repo (`feat:`, `fix:`, `docs:`)
- O clone de referência intocado está em `/tmp/folio-2025` (se não existir: `git clone --depth 1 https://github.com/brunosimon/folio-2025.git /tmp/folio-2025`)
